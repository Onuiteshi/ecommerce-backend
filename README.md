ECOMMERCE BACKEND SERVER

This is a simple ecommerce application that performs basic operations such as ;

- Viewing Products
- Adding to cart
- Checkout Products
- Viewing Profile
- Login
- Search Products

This project serves as the backend server for the mobile application,using Node js and GraphQL

Libraries used; 

- graphql
- express
- express graphql

HOW TO USE; 

Tos start project; 

    pull from repo
    npm i
    node server.js
